Range Control Platform

The Range Control Platform is designed to reduce manual judgement in the ranging process by introducing structured validation around store grade, space limits, stand capacity, product allocation, and budget control. The goal is to help users create more accurate plans, reduce over-allocation and under-allocation, and provide clearer visibility of validation breaches before decisions are finalised.

Key Features

- Build plans by branch / period / grade
- Apply grade-based ranging rules
- Validate stand capacity and product allocation
- Check category and budget constraints
- Surface warnings and validation breaches
- Support override workflows with audit visibility
- Generate reports for review and analysis
- Swap between local/demo data and BigQuery-backed data access
